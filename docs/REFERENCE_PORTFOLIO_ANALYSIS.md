# Reference Portfolio Analysis - Prateek Kashyap

## Portfolio Structure & Design

### Layout
- **Sidebar Navigation** (Left side - dark navy/teal)
  - Profile picture (circular, top)
  - Name: Prateek Kashyap
  - Title: Analytics & AI Consultant
  - Location: Singapore
  - Navigation buttons: Home, Experience, Skills, Portfolio, GitHub, Contact
  - Social icons at bottom (LinkedIn, Email)

- **Main Content Area** (Right side - light background)
  - Hero section with name, title, and professional photo
  - Professional summary/bio
  - Experience section with company, role, dates, location, achievements
  - Skills section (organized by category)
  - Portfolio/Projects section with descriptions
  - Contact section with form

### Design Elements
- **Color Scheme**: Dark navy/teal sidebar with light gray/white main content
- **Typography**: Clean, modern fonts
- **Spacing**: Well-organized sections with clear hierarchy
- **Interactive Elements**: Buttons, links, form inputs
- **Visual Hierarchy**: Important information emphasized with color and size

### Content Sections
1. **Hero Section**
   - Name (large, prominent)
   - Title/Role (highlighted in green/teal)
   - Professional photo (right side)
   - Bio/Summary (2-3 lines)
   - Call-to-action buttons (Connect on LinkedIn, Email Me)

2. **Experience Section**
   - Company name
   - Job title
   - Dates (From - To)
   - Location
   - 3-4 bullet points with achievements and metrics

3. **Skills Section**
   - Organized by category (Data & Analytics, Machine Learning & AI, etc.)
   - Skill tags/pills

4. **Portfolio/Projects Section**
   - Project title
   - Description
   - Technologies used
   - Metrics/results

5. **Contact Section**
   - Contact form (Name, Email, Message)
   - Social links
