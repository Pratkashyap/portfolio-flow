## Step 4: Questionnaire 📋

### What You'll See
- **Progress Bar**: "Step 2 of 4"
- **7 Questions**: Personalization form
- **Question Types**: Text inputs, radio buttons, dropdowns
- **Continue Button**: To proceed to template selection

### The 7 Questions

**Q1: What's your job title?**
- Type: Text input
- Example: "Data Analytics Leader", "Software Engineer", "Product Manager"
- Purpose: Personalizes portfolio tone

**Q2: Years of experience?**
- Type: Radio buttons
- Options: 0-2, 2-5, 5-10, 10+ years
- Purpose: Determines seniority level

**Q3: Your work style?**
- Type: Radio buttons
- Options: Collaborative, Independent, Data-Driven, Creative, Balanced
- Purpose: Influences messaging

**Q4: Key achievement?**
- Type: Text input
- Example: "Led analytics strategy driving 8% engagement uplift"
- Purpose: Highlights accomplishment

**Q5: Design preference?**
- Type: Radio buttons
- Options: Modern & Professional, Minimalist & Clean, Bold & Creative
- Purpose: Influences template choice

**Q6: Favorite color?**
- Type: Dropdown
- Options: Navy & Mint, Slate & Blue, Purple & Pink, Other
- Purpose: Customizes color scheme

**Q7: Your industry?**
- Type: Dropdown
- Options: Technology, Finance, Healthcare, Creative, Other
- Purpose: Tailors styling

## Step 7: Portfolio Generation & Deployment 🚀

### What Happens
When you click "Generate & Publish":

1. **Database Entry Created**
   - Portfolio data saved to database
   - User association created
   - Metadata stored

2. **Unique Slug Generated**
   - Format: `[first-name]-[last-name]-[user-id]`
   - Example: `prateek-kashyap-1`
   - Guaranteed unique

3. **Deployment to Manus**
   - Portfolio deployed to Manus infrastructure
   - Custom domain configured
   - SSL certificate applied
   - CDN optimization enabled

4. **Portfolio Goes Live**
   - Website accessible worldwide
   - Free hosting activated
   - Sharing enabled

### Processing Time
- Typical: 10-30 seconds
- Maximum: 60 seconds
- Status: Real-time progress indicator

### What You See

```
- Creating portfolio entry... ✓
- Generating unique URL... ✓
- Deploying to Manus... ✓
- Finalizing... ✓

Your portfolio is now live!
```

## Step 9: Sharing Options 📤

### What You'll See
- **Copy Link Button**: Copy URL to clipboard
- **Share on LinkedIn**: Pre-filled post
- **Share on Twitter**: Pre-filled tweet
- **Share via Email**: Pre-filled email
- **Dashboard Link**: To manage your portfolios

### Sharing Methods

**Method 1: Copy Link**
- Click "Copy Link" button
- URL copied to clipboard
- Paste anywhere you want

**Method 2: LinkedIn**
- Click "Share on LinkedIn"
- Opens LinkedIn with pre-filled post
- Add personal message
- Click "Post"

**Method 3: Twitter**
- Click "Share on Twitter"
- Opens Twitter with pre-filled tweet
- Customize message
- Click "Tweet"

**Method 4: Email**
- Click "Share via Email"
- Opens email client
- Pre-filled with portfolio link
- Add recipient and message
- Send

### Sharing Best Practices

**LinkedIn Strategy**
- Add personal message about your portfolio
- Mention key skills or achievements
- Use relevant hashtags (#Portfolio, #Hiring, #OpenToWork)
- Tag relevant connections

**Twitter Strategy**
- Keep message concise and engaging
- Use relevant hashtags
- Include call-to-action
- Tag industry leaders or recruiters

**Email Strategy**
- Personalize the message
- Explain why you're sharing
- Add context about your skills
- Include brief call-to-action

**Recruiter Strategy**
- Include portfolio link in resume
- Add to LinkedIn profile
- Share in job applications
- Include in cover letters
- Mention in interviews
